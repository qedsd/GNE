SYSTEM FEATURES

整合多项核心数据接口，提供一站式游戏辅助体验。

多角色管理

支持绑定多个 EVE 账号与角色。实时监控技能训练进度、资产变动、邮件及钱包余额，告别频繁切号的烦恼。

市场中心洞察

实时获取吉他等核心星系的市场订单数据。商品历史价格趋势图表化展示，助你敏锐捕捉跨区贸易商机。

智能星图导航

内置 2D/3D 战术星图。自动规避高危星系，提供安全/最短路线规划，并支持击杀数据叠加显示，确保航行安全。

本地情报预警

自动解析游戏内聊天日志，实时播报本地星系敌对目标出没信息，支持声音及弹窗双重警报，刷怪挖矿更安心。

深度数据百科

内置涵盖舰船、装备、蓝图及星系的详尽数据库。支持离线快速检索与属性对比，新伊甸万物皆在掌握。

国服/欧服双端支持

底层架构同时兼容 EVE 晨曦（国服）与 Tranquility（欧服）API 数据接口，一次配置，双服畅游。

UI GALLERY

专为暗黑环境设计的现代化 WinUI 3 界面。
